== Description ==

A block to add an estimated reading time of a post.

== Installation ==
1. Upload the `post-reading-time-block` folder to your `/wp-content/plugins/` directory or alternatively upload the post-reading-time-block.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Search for the block within the Gutenberg block editor.
